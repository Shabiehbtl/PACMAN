# Pacman Game

A classic, retro-style **Pacman game** built from scratch using **Python**. 

Navigate the maze, eat all the pellets, and avoid the ghosts to get the highest score!

## Features

* **Classic Mechanics:** Retro movement and map layout.
* **Smart Ghosts:** AI logic for chasing the player.
* **Power Pellets:** Eat special dots to hunt ghosts.
* **Live Score:** Real-time tracking of your points.
